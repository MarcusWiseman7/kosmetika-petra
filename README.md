# kosmetika-petra

Nuxt 4 site for Kosmetika Petra.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Build and Preview for Production

```sh
npm run type-check
npm run build
npm run preview
```

Production runs as Node SSR via `.output/server/index.mjs` (see `ecosystem.config.cjs` for the PM2 process config).
